# JUNCOMMUNITY
## 게시판 구현



## 1. 사용 기술
  * Spring-Boot (SDK 1.8)
  * JPA
  * Mysql
  * BootStrap
  * lombok
  * mybatis
  * thymeleaf
  * spring web
  
## 2. 주요 기능
  * 게시글 등록, 수정, 삭제
  * 페이징 처리
  * 게시글 검색
